Don't Starve is definitely a "simulation" game...

Don’t Starve is an action-adventure survival simulation game for PC, PS4, Xbox One, iOS and Android. The game was developed by Klei Entertainment and was originally released for the PC in 2013.

The goal of the game is to survive, this is done by maintaining hunger, sanity and health. The player has to collect resources, craft tools and build shelter in order to survive. The game has many advanced mechanics including farming and crafting.

Don’t starve is a really fun game, but it is extremely difficult if you don’t know what you’re supposed to do.
